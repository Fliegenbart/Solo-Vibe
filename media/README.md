## File naming convention

- Demo screenshots: `demo-[step]-[action].png`
- Demo animations: `demo-[step]-[action].gif`
- Diagrams: `diagram-[topic].png`
- Social/marketing: descriptive name, no prefix

## Guidelines

- GIFs: max 5 MB, 15 fps, optimized with gifsicle
- Screenshots: PNG, 2x resolution for retina
- Social preview: exactly 1280x640px (GitHub requirement)
- No text smaller than 16px in images (readability)
- Use the product's actual UI, not mockups, once available
