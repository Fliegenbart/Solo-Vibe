## Kernbotschaften

### 1. Dein Code. Dein Server. Deine Kontrolle.

Du hast mit AI etwas gebaut — jetzt gehört es dir. Solo-Vibe bringt deinen Code auf deinen eigenen Server. Kein Hosting-Anbieter dazwischen. Kein Abo, das deinen Code als Geisel hält. Export jederzeit, ohne Kleingedrucktes.

### 2. Live in einem Klick.

Code hochladen. Server verbinden. Live schalten. Solo-Vibe kümmert sich um den Rest: übertragen, starten, prüfen. Wenn etwas nicht funktioniert, sagt dir Solo-Vibe warum — in Worten, die du verstehst.

### 3. Jeder Stand bleibt erhalten.

Jede Version deines Projekts wird gespeichert. Wenn ein Update Probleme macht, setzt du in Sekunden auf den vorherigen Stand zurück. Kein Risiko, kein Datenverlust. Probiere aus, ohne Angst.

### 4. Kein Git. Kein Terminal. Kein Fachchinesisch.

Solo-Vibe spricht deine Sprache. Statt Commits, Branches und Pull Requests gibt es Stände, Veröffentlichungen und einen Verlauf. Technik wirkt im Hintergrund — du siehst nur, was du brauchst.

### 5. Offen. Nicht eingesperrt.

Dein Projekt ist ein normaler Ordner mit normalem Code. Kein proprietäres Format, keine Solo-Vibe-Abhängigkeit. Wenn du morgen woanders hin willst, lädst du dein Projekt herunter und gehst. Wir machen es dir leicht zu bleiben, nicht schwer zu gehen.
